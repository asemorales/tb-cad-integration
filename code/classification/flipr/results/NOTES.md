# Provenance of these results

These files are transcribed from the FlipR coauthor's reported evaluation:
the metrics table in `../README.md` and the per-class evaluation screenshot
from the original run. The trained checkpoint was not retained, so they
cannot be regenerated by inference; reproducing them requires retraining
via `../run.sh`. The confusion matrix is the unique reconstruction implied
by the per-class supports (570/570/120) and sensitivities. No ROC-curve
artifact was saved by the original eval harness.
